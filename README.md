# Credent front-end

## Requirements

- node js
- yarn

### Install node js and yarn

```bash
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install --lts
npm install --global yarn
```

## Commands

- `yarn run clean`
- `yarn run build`
- `yarn run start`
